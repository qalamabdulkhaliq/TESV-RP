# Frostfall Roleplay — Discord Structure

---

## INFORMATION
| Channel | Purpose |
|---------|---------|
| `#welcome` | Server description, quick links, how to get started |
| `#rules` | Full server rules, linked to pinned post |
| `#lore-primer` | IC lore narrations — setting, factions, history |
| `#faq` | Common questions answered |
| `#patch-notes` | Game mode updates and changes |
| `#server-status` | Up/down announcements |

---

## APPLICATIONS
| Channel | Purpose |
|---------|---------|
| `#how-to-apply` | Instructions, requirements, form links |
| `#application-status` | Staff posts status updates, no discussion |
| `#accepted` | Accepted character announcements |

**Threads or tickets used for individual applications — not visible in main server.**

|---------|---------|

## IN-CHARACTER BOARDS
These channels are written in-character. No OOC text.

| Channel | Purpose |
|---------|---------|
| `#bounty-board` | Hold guards and Jarls post bounties |
| `#courier-board` | Public notices, letters, trade announcements |
| `#property-listings` | Available properties posted by Stewards |
| `#wanted-and-missing` | IC missing persons, wanted posters |
| `#trade-and-commerce` | Player-run business advertisements, trade offers |

---

## WORLD EVENTS
| Channel | Purpose |
|---------|---------|
| `#world-events` | Staff-posted IC events and developments |
| `#rumour-mill` | Player-posted IC rumours, hearsay, whispers |
| `#the-holds` | IC travel and scene-setting posts by hold |

## THE NINE HOLDS
One category per hold. Each contains:

| Channel | Purpose |
|---------|---------|
| `#[hold]-ooc` | OOC coordination for that hold's players |
| `#[hold]-ic-board` | IC notices, postings, announcements specific to that hold |

**Holds:**
- Whiterun Hold
- Eastmarch
- The Rift
- The Reach
- Haafingar
- The Pale
- Falkreath Hold
- Hjaalmarch
- Winterhold

---

## FACTIONS
Private categories. Visible only to members of that faction and staff.

Each faction category contains:

| Channel | Purpose |
|---------|---------|
| `#[faction]-ooc` | OOC coordination |
| `#[faction]-ic` | IC communication within the faction |
| `#[faction]-orders` | Posted by leadership — assignments, directives |
| `#[faction]-bbb` | Pinned BBB document (Benefits, Burdens, Bylaws) |

**Faction categories:**
- Imperial Garrison
- Fourth Legion Auxiliary
- Aldmeri Dominion (Thalmor)
- The Companions
- College of Winterhold
- Thieves Guild
- Bards College
- Confederation of Temples
- Forsworn
- Stormcloak Underground
- East Empire Company

## ROLES

### Access Roles (assigned on application approval)
| Role | Grants access to |
|------|-----------------|
| `Citizen` | General channels, hold channels, IC boards |
| `[Faction Name]` | Faction private category |
| `Hold Leader` | Leadership category |
| `Staff` | Staff category |

### Display Roles (cosmetic, faction/hold identity)
- One per Hold (e.g. `Whiterun`, `Eastmarch`)
- One per Faction (e.g. `Imperial Garrison`, `Thalmor`)
- `Jarl` — Hold heads
- `Steward` — Hold administrative officers
- `Fourth Legion Auxiliary` — vetted lieutenant's company

### Notification Roles (opt-in)
- `@Events` — world event announcements
- `@Patches` — game mode update pings
- `@LaunchAlert` — launch event ping (one-time use)
